**2. Create `src/app/(site)/equipos/page.tsx`** — server component (NO `'use client'`).

```typescript
import { client } from '@/sanity/client'
import { EQUIPOS_LIST_QUERY } from '@/sanity/queries/equipos'
import { EquipmentGrid } from '@/components/equipos/equipment-grid'

export default async function EquiposPage() {
  const equipos = await client.fetch(EQUIPOS_LIST_QUERY).catch(() => [])

  return (
    <section className="px-4 py-12 max-w-7xl mx-auto">
      <h1 className="text-3xl font-bold text-brand-text mb-8">Catalogo de Equipos</h1>
      <EquipmentGrid equipos={equipos} />
    </section>
  )
}
```

The server component fetches all equipment at request time and passes the array to the client component. The client handles all filter/search state. Fetch failure returns empty array (graceful degradation when Sanity is not configured).
