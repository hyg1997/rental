## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding |
| sm | 8px | Compact spacing between nav items, icon-to-label gap |
| md | 16px | Default element spacing, nav horizontal padding |
| lg | 24px | Navbar vertical padding, section padding within Footer columns |
| xl | 32px | Footer section block spacing |
| 2xl | 48px | Major section breaks between Footer regions |
| 3xl | 64px | Page-level top/bottom padding on placeholder page |

Exceptions: Touch targets (hamburger button, nav links on mobile, WhatsApp button) must be minimum 44x44px per WCAG 2.5.5. This is an interaction size minimum, not a spacing token. Implement via `min-w-[44px] min-h-[44px]` on interactive elements.

Source: Standard 8-point scale (default). 44px touch target exception from UX-01 (mobile-first) and WCAG.

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 (regular) | 1.5 |
| Label | 14px | 400 (regular) | 1.4 |
| Heading | 20px | 700 (bold) | 1.2 |
| Display | 28px | 700 (bold) | 1.15 |

All sizes use Inter from `next/font/google`. Only two weights are active: 400 and 700. No intermediate weights (500, 600) are declared for this phase.

Source: UX-02 ("tipografía bold"), RESEARCH.md Pattern 4 (`--font-heading` and `--font-body` both Inter). Sizes are default shadcn/ui conventions for a professional B2B site at this phase scope.

## Color

All values are OKLCH — defined as CSS custom properties in `globals.css` via `@theme inline`. No `tailwind.config.js` is used.

| Role | OKLCH Value | Hex Approximate | Usage |
|------|-------------|-----------------|-------|
| Dominant (60%) | `oklch(0.12 0.01 260)` | #0d0d14 | Page background (`bg-brand-bg`) |
| Secondary (30%) | `oklch(0.18 0.01 260)` | #16161f | Navbar background, Footer background, card panels (`bg-brand-surface`) |
| Accent primary (10% — red) | `oklch(0.55 0.22 25)` | #c0392b approx | Primary CTA buttons, active nav indicator, WhatsApp button border ring |
| Accent secondary (yellow) | `oklch(0.82 0.18 85)` | #f1c40f approx | Hover state on accent-red elements, decorative highlights, icon accent on features |
| Text | `oklch(0.96 0.01 260)` | #f0f0f5 | All body text, nav links, footer text (`text-brand-text`) |
| Destructive | `oklch(0.55 0.22 25)` | same as accent-red | Destructive actions only (not used in Phase 1 user-facing UI; reserved) |

Accent reserved for:
- "Solicitar Cotización" CTA button (Navbar, desktop and mobile)
- WhatsApp floating button ring/icon color
- Active/current page indicator in nav links
- Hover state on nav links (text color shift to accent-red)

Yellow accent reserved for:
- Hover highlight on red-accented interactive elements only
- No standalone yellow blocks at this phase

Source: RESEARCH.md Pattern 4 (exact OKLCH values). UX-02 ("colores oscuros, acentos rojos/amarillos"). rdrental.com.pe design reference.

## Interaction Contract

### Navbar (Responsive)

**Desktop (md and above):**
- Horizontal nav links: Inter 14px weight 400, color `text-brand-text`, hover shifts to `text-brand-red`
- Active link: `text-brand-red` with a 2px bottom border in `brand-red`
- "Solicitar Cotización" button: `bg-brand-red` text white, hover `bg-brand-yellow text-black`, 8px vertical / 16px horizontal padding, rounded-md
- Logo: left-aligned, max-height 40px

**Mobile (below md — 360px is the test target):**
- Logo left-aligned
- Hamburger icon (Menu from lucide-react): right-aligned, 44x44px touch target, `text-brand-text`
- Sheet slides in from right, 256px wide, `bg-brand-surface`
- Nav links stacked vertically inside Sheet, 48px min-height each (touch target compliance)
- "Solicitar Cotización" button at bottom of Sheet menu

**Hamburger button label:** `aria-label="Abrir menú"` (Spanish, per project language)

### Footer

- Three columns on desktop (contact info / navigation links / social links), single column stacked on mobile
- All data fetched from `siteSettings` Sanity document — no hardcoded contact values
- Copyright text: Inter 14px weight 400, `text-brand-text` at 60% opacity

### WhatsApp Button

- Fixed position: bottom-right, `bottom-6 right-6` (24px from edges)
- 56x56px circle button, `bg-green-600` (WhatsApp brand green, not brand-red)
- On mobile: must not obscure Navbar or any primary CTA — test at 360px
- `aria-label="Contactar por WhatsApp"` (Spanish)
- Opens `https://wa.me/{whatsappNumber}` from siteSettings, target `_blank`

### Placeholder Homepage

- Full viewport height (`min-h-screen`), centered content
- Heading: Display (28px, weight 700) — "Bienvenido a Testing Calibrations S.A.C."
- Body: 16px weight 400 — "Sitio en construcción. Vuelve pronto."
- `bg-brand-bg` background
